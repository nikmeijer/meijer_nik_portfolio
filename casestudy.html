    Cinema 4D Earbuds

    The goal of this project was to create a pair of wireless earbuds with a charging case in Cinema 4D. and make an animation for a promotional video.

    After searching the internet for inspiration I decided the Sony Linkbuds would provide an interesting challenge to build out, as they have a unique open loop shape to them.

    Building the model proved to be less of a challenge than I anticipated. To build out the loop, I set up the framework for the donut shape using circles and the put them in a loft to link it all. The only trouble I ran into was the heirarchy of layers, and arranging the circles in a way that they filled out and closed the loop

    One other challenge I faced was linking all the objects into one without creating abrupt edges. I was able to sort this out by creating an extra piece to go between the loop and the shell of the buds, which created a fade/crossover.

    Once the model was built out, I built the shell using a collider, I made some mock-up internals for the earbuds, and created materials for it to simulate a glossy plastic.

    The final step was animating. Once I had all the materials set up, and lighting was set, I was ready to keyframe, and then finally rendering out with redshift to a png sequence which I then arranged in After Effects, and finally, I had my result.
